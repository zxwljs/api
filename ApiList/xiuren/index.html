    该API返回图片内容

    请求

    URL

    https://c9aa2c19-16c5-4624-82fb-5fbf20fefb06.bspapp.com/xiuren

    方式

    ANY
    GET/POST/HTTP等

    响应

    参数说明

    名称 类型 说明 描述
    msg String success时为成功 返回的信息
    code Num 200时为正常 状态代码
    img String --- 图片直链

    示例

    {
    "msg": "success",
    "code": 200,
    "img": "https://t.xiurenji.net/uploadfile/202005/18/78165633253.jpg"
}
